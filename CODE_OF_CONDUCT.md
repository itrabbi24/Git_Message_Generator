# Code of Conduct

## Our Standard

This project aims to provide a respectful, constructive, and professional environment for everyone.

Examples of expected behavior:

- Be respectful in discussions
- Give technical feedback clearly and constructively
- Focus on improving the project
- Assume good intent while reviewing code and ideas

Examples of unacceptable behavior:

- Harassment or abusive language
- Personal attacks
- Bad-faith disruption of discussions or contributions
- Publishing private information without permission

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, or other contributions that do not align with this code of conduct.
