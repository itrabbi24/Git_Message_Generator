# Contributing

Thank you for contributing to `Git Message Generator By ARG RABBI`.

## Development Setup

```bash
npm install
npm run build
```

Open the project in VS Code and press `F5` to run the extension in an Extension Development Host.

## Contribution Guidelines

- Keep changes focused and reviewable.
- Follow the existing TypeScript structure and naming conventions.
- Prefer small commits with clear conventional commit messages.
- Update documentation when user-facing behavior changes.
- Test the extension manually against staged Git changes before submitting.

## Pull Requests

Please include:

- a short summary of the change
- why the change is needed
- screenshots or examples if the UI or output changed
- any limitations or follow-up work

## Reporting Issues

When opening an issue, include:

- VS Code version
- operating system
- extension version
- steps to reproduce
- expected result
- actual result
- sample staged diff if relevant

## Code Style

- TypeScript with strict settings
- Clear naming over clever code
- Minimal comments unless logic needs explanation
- No unrelated refactors in the same change
